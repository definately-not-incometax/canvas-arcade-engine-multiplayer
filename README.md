# canvas-arcade-engine-multiplayer
A browser-based 2D game engine built with HTML5 Canvas and vanilla JavaScript, featuring a custom game loop, real-time input handling, collision detection, and modular architecture. Designed to demonstrate core game development concepts such as physics simulation, rendering pipelines, and state management, all running on a local server environment.
